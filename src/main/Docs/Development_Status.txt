0. Executive Summary
--------------------
EyeAI fuses deep-learning agents, emergent narrative, and The Pit minigame ecosystem to create the **first fully-autonomous MMO dungeon arena** on Paper 1.20.6.  Every mob, quest and economy ripple is driven by _RL-trained FakePlayers_ and the **Mem0** long-term memory graph.

The project is **code-complete for Alpha 2**: the server starts, agents fight, players trade, and all GUIs are live.  Focus now shifts to world-scale content generation, Mem0 pipelines, and anti-cheat ML.
